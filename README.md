# E-Commerce using Geo-Navigation
This user-friendly app is designed to enhance your shopping experience by providing a convenient way to locate shops that offer a variety of grocery items. Our app not only helps you find shops near you, but also displays the prices of different items along with any available discounts. This project consists of two apps(EC-GeoCustomer and EC-GeoOwner)

# EC-GeoCustomer
E-Commerce using Geo-Navigation Customer is the android application abbreviated as EC-Geo Customer. This is the branch of E-Commerce using Geo-Navigation where customer can search the shops where particular product is available.Here customer can compare rate and can also place order online. If there are any shortcomings of products at home one can easily get the shops where the product is available. As a result it is less time consuming.

![ic_launcher](https://github.com/1405yuga/EC-GeoCustomer/assets/82303711/8347173c-939c-4c8e-bc67-8b2861533ba6)

## Guide
### Login page : 

Customer can enter Email and password generated while creating account to login the application. If customer is new they can click on **Sign Up** to create new account.

<img src="https://github.com/1405yuga/EC-GeoCustomer/assets/82303711/7103bf06-c6ae-4eb4-a998-b6c486393d11" width="350" height="700">

### Search : 

In search page one can check the availability of products in the shop. Initially customer's location is displayed on the map with red marker.

<img src="https://github.com/1405yuga/EC-GeoCustomer/assets/82303711/40f44414-910c-4e64-8cf1-86b5adafb279" width="350" height="700">

     
At top in search bar customer can search the products and list of the products are displayed in realtime.

<img src="https://github.com/1405yuga/EC-GeoCustomer/assets/82303711/deeb26fc-c94c-4e71-9697-b8336cb42ec8" width="350" height="700">


On Clicking particular product, shops where it is available are displayed on map with yellow marker. Each marker can display information such as Shop's name and rate.

<img style="margin-right: 2.5rem" align="left" src="https://github.com/1405yuga/EC-GeoCustomer/assets/82303711/c9292040-6cc6-4b3d-a031-0bdff6fbc2fa" width="350" height="700">

<img src="https://github.com/1405yuga/EC-GeoCustomer/assets/82303711/a0608312-8058-43e9-bda0-5365627c5cfb" width="350" height="700">


When long pressed on information window, Order placing dialog is displayed. In this, customer can enter the quantity required and buy.

<img src="https://github.com/1405yuga/EC-GeoCustomer/assets/82303711/c756c25d-c248-4083-b5e9-525cc733c384" width="350" height="700">


Later confirmation page is generated. At bottom of this page, other similar products are recommended. When confirmed the order the app navigates to the payment applications such Paytm, Google pay etc. On paying the amount order is placed.
          
<img src="https://github.com/1405yuga/EC-GeoCustomer/assets/82303711/d3c6bdfb-12b6-4fd5-8e7c-884b9a4679f5" width="350" height="700">
